# Declarative

_DSL for nested schemas._

[![Gem Version](https://badge.fury.io/rb/declarative.svg)](http://badge.fury.io/rb/declarative)

# Overview

Declarative allows _declaring_ nested schemas.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'declarative'
```
